  request at 25 MB — but line 70 allows single files up to 50 MB, and the
  endpoint accepts multiple files. Five 6-MB JPEGs → 413 "Upload too large".
  A single 30-MB file passes the per-file check but can never arrive.
  FIX (match the per-file policy, like photos/upload.ts does):
    const tooLarge = rejectOversizedRequest(context.request, 50 * 1024 * 1024 * 4);
    // or: import MAX limits from upload-policy and use maxFiles * maxBytes
  ...and enforce a max file count explicitly:
